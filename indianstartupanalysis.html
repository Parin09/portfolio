Grocery Store Case Study

Exploratory Data Analysis of startup ecosystem in India based on dataset obatined from Kaggle.

Parameters in analysis include investors, funding rounds, amount invested, top cities and industry verticals. Intended audience of this analysis include Venture Capitalist Firms,Private Investors and Entrepreneurs.

Primary objective is to demystify the data and present the findings in a feasible and effective format for the general audience to interpret.

Data cleaning performed using Python and analysis using Tableau
